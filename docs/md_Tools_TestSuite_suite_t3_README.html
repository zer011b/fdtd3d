Description

This is test with comparison of exact solution with numerical one for 3D mode.

Equations

1 dEx/dt = 1/eps0 * (dHz/dy - dHy/dz - Jx)
2 dEy/dt = 1/eps0 * (dHx/dz - dHz/dx - Jy)
3 dEz/dt = 1/eps0 * (dHy/dx - dHx/dy - Jz)
4 
5 dHx/dt = 1/mu0 * (dEy/dz - dEz/dy - Mx)
6 dHy/dt = 1/mu0 * (dEz/dx - dEx/dz - My)
7 dHz/dt = 1/mu0 * (dEx/dy - dEy/dx - Mz)

Exact solution

1 Ex = c^2 * t * (y^2 + z^2)
2 Ey = c^2 * t * (x^2 + z^2)
3 Ez = c^2 * t * (x^2 + y^2)
4 
5 Hx = c^2 * eps0 * t * (y^2 + z^2)
6 Hy = c^2 * eps0 * t * (x^2 + z^2)
7 Hz = c^2 * eps0 * t * (x^2 + y^2)
8 
9 Jx = -c^2 * eps0 * (2 * t * (z - y) + y^2 + z^2)
10 Jy = -c^2 * eps0 * (2 * t * (x - z) + x^2 + z^2)
11 Jz = -c^2 * eps0 * (2 * t * (y - x) + x^2 + y^2)
12 
13 Mx = -c^2 * (2 * t * (y - z) + eps0 * mu0 * (y^2 + z^2))
14 My = -c^2 * (2 * t * (z - x) + eps0 * mu0 * (x^2 + z^2))
15 Mz = -c^2 * (2 * t * (x - y) + eps0 * mu0 * (x^2 + y^2))

Numerical solution

Case of dt = 0.5 * dx / c, i.e. Courant number set to 0.5. Numerical solution should match exact solution (with floating point accuracy).

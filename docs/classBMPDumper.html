Detailed Description

template<class TCoord>
class BMPDumper< TCoord >

Grid saver to BMP files. Template class with coordinate parameter.

NOTE: ".bmp" dumper/loader can't reproduce field values precisely. Consequent dump to and load from ".bmp" file of grid will change grid values.
